# Game Content Directory

This directory is packaged with the game executable as `content/` next to `StarArkGame.exe`.

Structure convention (created as needed):
- `models/` — `.obj`, `.fbx`, `.gltf` files
- `textures/` — albedo/normal/MRA/emissive maps
- `shaders/` — custom GLSL (overrides engine built-ins)
- `scenes/` — JSON scene files (Phase M2+)
- `materials/` — JSON material files (Phase M2+)
